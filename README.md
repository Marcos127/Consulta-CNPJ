# Consulta de Empresas por CNPJ (Python)

Aplicação desktop desenvolvida em Python para consulta automática de empresas utilizando o CNPJ.

O sistema busca informações da empresa através da API pública BrasilAPI e permite armazenar os dados em banco de dados local.

# Tecnologias utilizadas

- Python
- Tkinter (interface gráfica)
- Requests (requisições HTTP)
- SQLite (banco de dados)
- ReportLab (geração de PDF)
- BrasilAPI (consulta de CNPJ)

# Funcionalidades

- Consulta automática de dados de empresas pelo CNPJ
- Preenchimento automático das informações
- Armazenamento dos dados em banco SQLite
- Exportação de relatórios
- Interface gráfica simples para utilização

# Objetivo do projeto

Este projeto foi desenvolvido para praticar:

- Consumo de APIs
- Desenvolvimento de aplicações desktop em Python
- Manipulação de banco de dados
- Geração de relatórios em PDF
